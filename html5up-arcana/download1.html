JAVA DEVELOPMENT KIT (JDK)

Java Development Kit (JDK) merupakan sebuah perisian yang menyediakan platform untuk mengkod dan mejalankan bahasa perisian JAVA. Ia termasuk JAVA Runtime Environment (JRE) ataupun persekitaran larian JAVA, sebuah penterjemah/muat naik (java), kompiler(javac), penyimpan(jar), penjana dokumen (javadoc), dan lain-lain alat yang digunakan dalam pengembangan Java

NETBEANS

NetBeans merupakan sebuah projek sumber terbuka yang didedikasikan untuk menyediakan produk pengembangan perisian(Netbeans IDE dan Netbeans Platform) yang dikhususlan untuk membantu pengaturcara, pengguna dan ahli bisnes yang bergantung kepada Netbeans sebagai asas kepada produk mereka; khususnya untuk membenarkan mereka untuk membangunkan produk dengan lebih cepat, cekap dan mudah dengan menggunakan kekuatan platform Java dan lain-lain standard relevan industri
